# CHARTERS

Canonical index of active arifOS charter files.

## Core
- `federation.charter.json`: federation-wide authored truth for trust boundaries, topology, and governance.
- `CONFIG/sovereignty.charter.json`: machine-readable sovereignty charter.
- `docs/sovereignty.charter.json`: documentation copy of the sovereignty charter.

## Deployment
- `deploy/stack.charter.json`: deploy-time pinned refs, images, and hold state.

## Kernel
- `config/charter/kernel.charter.yaml`: kernel capability and stage charter.
- `arifosmcp/tool_charter.py`: rich operational charter for the 13 canonical MCP tools.
- `arifosmcp/tools/charters/tool.charter.json`: runtime-readable tool safety charter.

## Spec
- `docs/reference/spec/mcp.charter.json`: MCP-facing authored spec charter.
- `docs/reference/spec/arif.charter.yaml`: A-RIF authored spec charter.

## Site
- `arifosmcp/sites/apex-dashboard/federation.charter.json`: Observatory-authored federation charter for public rendering.

## Naming Rule
- Use `charter` for authored declarative truth.
- Keep `server.json`, `registry.json`, `agent.json`, `server-card.json`, and other protocol discovery terms unchanged.
