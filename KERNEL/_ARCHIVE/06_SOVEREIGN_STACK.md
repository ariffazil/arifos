# 06_SOVEREIGN_STACK.md — The 7-Organ Architecture

The Intelligence Kernel is structured as a vertical stack of specialized organs.

- **000 (INIT):** The Airlock. Authentication and session ignition.
- **111-444 (AGI):** The Mind. Logical analysis (Math Agent).
- **555 (PHOENIX):** The Heart. Associative memory and empathy (Heart/Safety).
- **666 (HEART):** Ethical Safety. Conflict resolution.
- **777 (APEX):** The Soul. Final consensus and Genius calculation.
- **888 (FORGE):** The Hands. Material execution (Code Agent).
- **999 (VAULT):** The Memory. Immutable Ledger.

Each organ must sign the `Intent Envelope` before the message moves to the next layer.
