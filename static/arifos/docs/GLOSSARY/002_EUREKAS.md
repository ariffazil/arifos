## Eureka 2 — Reversibility Index
```text
R = 1.0 fully reversible
R = 0.7 reversible with backup
R = 0.3 costly rollback
R = 0.0 irreversible
```
R < 0.3 → 888 HOLD · 0.3 ≤ R < 0.7 → judge required · R ≥ 0.7 → proceed if authorized.
