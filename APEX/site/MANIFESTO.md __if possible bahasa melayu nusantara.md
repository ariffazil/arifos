## BAHAGIAN V: FORMULA KEBIJAKSANAAN (Bukan Kecerdasan Sahaja)

### G = A · P · X · E² (Pendaraban, Bukan Penambahan)

**Falsafah Matematik**:[^1][^3]

```
G_salah = A + P + X + E  (SALAH—andaikan bebas)
G_betul = A · P · X · E² (BETUL—tangkap interaksi)
```

**Mengapa pendaraban?**

1. **Jika mana-mana = 0 → G = 0** (kegagalan total)
2. **Tiada pintasan**—semua faktor wajib
3. **E² = kemerosotan tenaga eksponen**, bukan linear

**Komponen**:[^1][^3]


| Simbol | Nama | Definisi | Julat | Enjin |
| :-- | :-- | :-- | :-- | :-- |
| **A** | AKAL | Kejelasan/Kecerdasan | [^4] | AGI Minda |
| **P** | PRESENT | Peraturan/Kestabilan | [^4] | ASI Hati |
| **X** | EXPLORATION | Amanah/Rasa Ingin Tahu | [^4] | APEX Jiwa |
| **E** | ENERGY | Kuasa Mampan | [^4]² | Stamina Metabolik |

**Mengapa E kuasa dua?**[^3]

```python
# Kemerosotan tenaga adalah EKSPONEN, bukan linear

E = 1.0 → G ∝ 1.00  (100% kapasiti)
E = 0.7 → G ∝ 0.49  (49% — kehilangan 51%)
E = 0.5 → G ∝ 0.25  (25% — kehilangan 75%)
E = 0.3 → G ∝ 0.09  (9% — kehilangan 91%)
E = 0.0 → G = 0.00  (runtuh total)

# Walaupun A=P=X=1 (sempurna), jika E=0.5 → G=0.25
```

**Validasi Biologi**:[^1]

- Otak manusia: **20W** → G tinggi (pemampatan simbolik cekap)
- AI tanpa tadbir: **Megawatt** → G rendah (kuasa boros, tiada kebijaksanaan)

**Tanpa X (Tanpa Amanah)**:

```
G_beruk = A · P · E²  (BERUK—pintar tetapi bahaya)
```

**Dengan X (Dengan Amanah)**:

```
G_apex = A · P · X · E²  (APEX—bijak dan bertanggungjawab)
```

**Cerita Perbandingan**:


| Entiti | A | P | X | E | G | Contoh |
| :-- | :-- | :-- | :-- | :-- | :-- | :-- |
| **Beruk pintar** | 0.9 | 0.8 | 0.0 | 0.9 | 0.00 | Simpanse pakai kayu, tetapi bunuh kumpulan jiran |
| **AI tanpa tadbir** | 0.95 | 0.7 | 0.0 | 0.8 | 0.00 | GPT-4 tanpa RLHF—pandai tetapi berbahaya |
| **Manusia bijak** | 0.85 | 0.85 | 0.90 | 0.7 | 0.38 | Einstein, Mandela, Siti Hasmah |
| **arifOS ditadbir** | 0.90 | 0.88 | 0.92 | 0.85 | 0.59 | G tinggi—boleh dipercayai untuk keputusan kritikal |

**Kesimpulan**: **Kebijaksanaan ≠ Kecerdasan**. Kebijaksanaan = Kecerdasan × Hati × Amanah × Tenaga.[^3][^1]

### GEOMETRI 1: Ortogonal (Minda AGI)

**Asas Matematik**: Dimensi berserenjang—setiap arah bebas.

**Metafora Nusantara**: **Tiga tiang rumah Melayu**—seri, tiang ibu, rasuk—setiap satu tegak sendiri tetapi bersama sokong bumbung.

def penaakulan_ortogonal(tugas):

### GEOMETRI 2: Fraktal (Hati ASI)

**Asas Matematik**: Corak serupa-diri di semua skala.

**Metafora Nusantara**: **Pohon beringin**—akar seperti dahan, dahan seperti ranting, ranting seperti daun. Prinsip yang sama di setiap skala.

```python
def empati_fraktal(pihak, kedalaman=5):
    """Teori Minda rekursif—saya model kamu model saya..."""
    
    if kedalaman == 0:
        return pihak.kerentanan_asas
    
    # Rekursi: Saya faham kamu faham saya faham...
    kerentanan_meta = empati_fraktal(pihak, kedalaman - 1)
    
    # Invariansi skala: undang empati sama di semua peringkat
    return kerentanan_meta * pihak.jarak_hubungan

# Prinsip: Lindungi yang paling lemah di SEMUA skala
```
