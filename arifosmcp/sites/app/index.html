What this is

arifOS Command Center MCP is the interactive cockpit for governed MCP execution. It renders inside compatible MCP hosts and provides chambers for session state, ops vitals, constitutional verdicts, forge dry-runs, gateway handshakes, and vault review.

The app does not bypass governance. It uses the same 13-tool public surface, the same human authority model, and the same audit-first runtime contract.

How to Connect

If your host supports MCP Apps, open command_center() from the MCP server.

If your host does not support MCP Apps, the same backend tools return text and JSON fallback.

  • Level 1: Plain MCP — text + JSON results from every tool
  • Level 2: MCP with resources — Prefab UI resource at command_center()
  • Level 3: MCP Apps host — interactive cockpit with buttons, forms, and live state

Chambers

  • Session — Constitutional identity, stage, lane, authority
  • Ops — Thermodynamic vitals (G, ΔS, Ω, Ψ)
  • Judge — Constitutional verdicts (SEAL / SABAR / HOLD / VOID)
  • Forge — Dry-run manifest simulation only
  • Gateway — Simulated cross-agent handshake
  • Vault — Mock audit entries and dry-seal preview
