The constitutional AI governance engine. Every inference passes through 13 epistemic floors, a metabolic pipeline (000→999), and produces a governed response with machine, governance, and intelligence envelopes.

What is arifosMCP?

arifosMCP is a Model Context Protocol (MCP) server that wraps AI inference in constitutional governance. It enforces truth floors, tracks epistemic uncertainty via 3E telemetry, and produces auditable verdicts: SEAL SABAR VOID 888_HOLD

Key Concepts

  • Constitutional Governance — Every response passes through 13 floors (F1–F13) that enforce truth, trust, empathy, and sovereign authority.
  • Metabolic Pipeline — Inference flows through stages 000_INIT → 111_SENSE → 222_REALITY → ... → 999_VAULT, with each stage responsible for a discrete epistemic function.
  • MGI Envelope — Every response is wrapped in a Machine / Governance / Intelligence triple envelope for full auditability.
  • 3E Telemetry — Three epistemic phases (Exploration, Entropy, Eureka) track the quality of reasoning in real time.
  • MCP Protocol — Standard JSON-RPC 2.0 over Streamable HTTP. Compatible with Claude Desktop, ChatGPT, Cursor, and any MCP client.

Verdicts

Every governed inference produces one of four verdicts:

SEALFully approved — all floors passedAll floor scores above thresholds
SABARPartial approval — soft floor violationsSome soft floors below threshold
888_HOLDMechanical pause — infrastructure issueTimeout, DNS failure, rate limit, etc.
VOIDConstitutional collapse — terminal rejectionHard floor violation (truth, auth, injection)
Key Invariant

Mechanical faults ALWAYS → 888_HOLD. NEVER → VOID. VOID is constitutional collapse only.

3E Telemetry

Every inference passes through three epistemic phases. The 3E framework tracks reasoning quality in real time.

Exploration

The initial phase where the system gathers information, explores the problem space, and builds an evidence base. Metrics include search breadth, source diversity, and grounding confidence.

Entropy

The reduction phase where information is compressed, contradictions resolved, and uncertainty systematically reduced. F4 (Clarity) enforces ΔS ≤ 0 — entropy must decrease. If entropy increases during this phase, the floor is violated.

Eureka

The crystallization phase where insights emerge, conclusions form, and the governed response takes shape. Quality is measured by the composite 3E score — a weighted aggregate of all three phases.

3E Composite Score

The 3e_composite field in the IntelligenceEnvelope provides a single float (0–1) representing overall reasoning quality. This feeds into F8 (Genius) — the governed intelligence floor requires ≥0.80.

VOID Memanjang

VOID Memanjang = progressive epistemic collapse. When 3E telemetry detects sustained entropy increase without Eureka crystallization, it triggers escalating warnings. Counter: 3E telemetry + F2 (Truth) + F7 (Humility) + mandatory verdicts prevent infinite collapse loops.

Add to your Claude Desktop configuration file (~/.claude/claude_desktop_config.json):
