Complete schema, type definitions, error codes, and integration patterns for the arifosMCP constitutional server. All responses are Pydantic v2 typed.

MGI Response Envelope
TypeScript · Pydantic v2

Every tool response wraps three typed layers. The outer Machine envelope captures runtime state, Governance captures constitutional verdicts, and Intelligence carries 3E telemetry and payload.

TypeScript interface
// Full MGI response envelope — every tool returns this
interface MGIResponse {
  machine: {
    ok:         boolean          // false on any fault
    fault_code: FaultCode | null // null when ok === true
    tool_name:  string
    latency_ms: number
    session_id: string
    version:    string           // semver "2026.03.x"
  }
  governance: {
    verdict:          Verdict        // SEAL | HOLD | VOID | EVOLVING
    floor_scores:     Record<FloorId, number>
    tri_witness_score:number         // W3 ∈ [0, 1]
    floor_breach:     FloorId | null
    hold_id:          string | null  // set when verdict === HOLD
  }
  intelligence: {
    three_e:  ThreeEState
    payload:  unknown          // tool-specific result
    delta_s:  number           // thermodynamic entropy Δ
  }
}

type Verdict  = "SEAL" | "HOLD" | "VOID" | "EVOLVING" | "GENESIS"
type FloorId  = "F1" | "F2" | "F3" | "F4" | "F5" | "F6" | "F7" | "F8" | "F9" | "F10" | "F11" | "F12" | "F13"
Python (Pydantic v2)
Pydantic v2
pydantic schema
from pydantic import BaseModel
from typing import Optional, Literal

class MachineEnvelope(BaseModel):
    ok:         bool
    fault_code: Optional[str] = None
    tool_name:  str
    latency_ms: float
    session_id: str
    version:    str

class GovernanceEnvelope(BaseModel):
    verdict:           Literal["SEAL","HOLD","VOID","EVOLVING","GENESIS"]
    floor_scores:      dict[str, float]
    tri_witness_score: float  # W3 ∈ [0, 1]
    floor_breach:      Optional[str] = None
    hold_id:           Optional[str] = None

class IntelligenceEnvelope(BaseModel):
    three_e:  ThreeEState
    payload:  dict
    delta_s:  float

class MGIResponse(BaseModel):
    machine:    MachineEnvelope
    governance: GovernanceEnvelope
    intelligence: IntelligenceEnvelope

Error handling pattern
async def safe_call(tool, **kwargs):
    result = await session.call_tool(tool, kwargs)
    r = MGIResponse.model_validate(result)

    if not r.machine.ok:
        match r.machine.fault_code:
            case "HOLD_REQUIRED":
                # Pause — await ratify_hold_state(hold_id=r.governance.hold_id)
                return await await_hold(r.governance.hold_id)
            case "FLOOR_BREACH" | "VAULT_INTEGRITY":
                # Constitutional fault — do not retry
                raise ConstitutionalViolation(r.governance.floor_breach)
            case x if x in RETRYABLE_CODES:
                return await retry_with_backoff(tool, **kwargs)
            case _:
                raise MCPFault(r.machine.fault_code)

    return r

class ExplorationTelemetry(BaseModel):
    sources_consulted: int     # number of sources hit
    depth_level:       int     # 1=surface, 3=deep dive
    breadth_score:     float   # ∈ [0, 1]

class EntropyTelemetry(BaseModel):
    uncertainty_index:    float  # ∈ [0, 1], lower = more certain
    contradiction_count:  int
    source_conflict_ids:  list[str]

class EurekaTelemetry(BaseModel):
    insight_delta:  float   # change from prior state
    novelty_score:  float   # ∈ [0, 1]
    crystallised:   bool    # True = stable insight reached

class ThreeEState(BaseModel):
    exploration: ExplorationTelemetry
    entropy:     EntropyTelemetry
    eureka:      EurekaTelemetry

    def is_complete(self) -> bool:
        return (
            self.exploration.sources_consulted > 0
            and self.eureka.crystallised
        )
Tri-Witness W3 Formula
Governance

W3 is the geometric mean of Human (H), AI (A), and Earth/Evidence (E) witness scores, weighted by their constitutional floor assignments.

w3.py
def compute_w3(
    s_H: float,  # Human approval score  (F13)
    s_A: float,  # AI coherence score    (F8)
    s_E: float,  # Evidence confidence   (F2)
    w_H: float = 0.4,
    w_A: float = 0.3,
    w_E: float = 0.3,
) -> float:
    return (w_H*s_H * w_A*s_A * w_E*s_E) ** (1/3)

# Verdict thresholds
# W3 >= 0.75 → SOVEREIGN  → SEAL permitted
# W3 >= 0.40 → EVOLVING   → HOLD issued, await ratify
# W3 <  0.40 → GENESIS    → VOID possible on breach
