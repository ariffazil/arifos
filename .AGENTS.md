## Quick Start

```bash
# Install (uv-based)
pip install -e ".[dev]"
# or
uv sync --extra dev

# Run tests
python -m pytest tests/ -q --tb=short

# Lint / Type check
ruff check .
mypy arifosmcp/

# Start MCP server
python -m arifosmcp.server       # HTTP + SSE
# or
python -m arifosmcp --stdio      # Stdio transport

# Health check
curl http://localhost:8080/health
```
