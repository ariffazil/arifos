## WELL Healthcheck Fix — Completed

**Problem:** WELL container showed `unhealthy` — Docker healthcheck was calling `/mcp` endpoint without `Accept: text/event-stream` header → always returned 406 → Docker marked unhealthy

**Root cause:** `docker-compose.well.yml` healthcheck:
```
test: ["CMD", "python3", "-c", "import urllib.request; urllib.request.urlopen('http://127.0.0.1:8083/mcp')"]
```
The MCP HTTP transport requires SSE accept header. Without it, returns 406.

**Fix applied:**
```
test: ["CMD", "python3", "-c", "import socket; s=socket.socket(); s.connect(('127.0.0.1', 8083)); s.close(); print('ok')"]
```
Uses TCP socket check instead. Container now `healthy`.
