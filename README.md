After passing through the 13 Floors, every action receives one verdict:

| Verdict | Meaning | Outcome |
|---------|---------|---------|
| **SEAL** (000) | Perfect constitutional alignment | Execute immediately |
| **COMPLY** (101–499) | Compliant, minor remediation noted | Execute with notes |
| **CAUTION** (500–899) | Compliant, soft Floor warnings | Execute, warnings logged |
| **HOLD** | Awaiting human decision | Pause — human notified |
