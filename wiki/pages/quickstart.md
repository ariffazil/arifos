## Common Questions

**Q: Do I need to self-host arifOS?**
A: For evaluation, use the hosted endpoint. For production with sensitive data, self-host.

**Q: What happens if a floor fails?**
A: Hard floors → VOID (blocked). Soft floors → CAUTION (proceed with warning) or 888_HOLD (escalate).

**Q: Can I modify the 13 Floors?**
A: F1, F2, F9, F10, F13 cannot be softened (constitutional invariants). Others can be adjusted via F13 protocol with human SEAL.

**Q: How much latency does governance add?**
A: ~50-200ms per action, depending on which floors are active.

**Q: Can AI agents override arifOS?**
A: No. The 888 Judge (human) holds final authority. AI agents operate within constitutional bounds.
