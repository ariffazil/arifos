# Agents and AAA Architecture

The **AAA (Agentic, API, AI, Apps) Surface Layer** represents the practical operational intelligence ring (The Body) in the arifOS Constitutional Federation.

## Ring 3 — THE BODY
Hosted at `aaa.arif-fazil.com`, this is the layer where smolagents execute and governed intelligence does real work. It interacts directly with the constitutional kernel (Ring 2) and the human sovereign (Ring 1).

### Integration Points
- **WAW (Workspace Agent Workspace)**: Agentic UI powered by React + TypeScript + OpenClaw.
- **AF.A.FORGE Portal**: Unified portal to the Trinity Architecture, MCP Runtime, and GEOX Earth.
- **GEOX Earth Witness**: Reality gatekeeper for geoscience AI.

## Agent Workflow
Agents operating within the AAA layer must comply with the 13 Constitutional Floors. They access tools via MCP Endpoints (VPS or Cloud FastMCP) and receive explicit verdicts (SEAL, COMPLY, CAUTION, HOLD, SABAR, VOID) before taking irreversible actions.
