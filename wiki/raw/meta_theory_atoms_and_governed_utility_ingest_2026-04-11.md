## Source Intent

Standardize a small set of meta-theory atoms and scalar metrics that can:

1. describe arifOS outputs in physics, math, and linguistic terms
2. drive a Karpathy-style autoresearch loop with one ratcheting scalar

## Proposed Meta-Theory Atoms

- **GL / Gödel Lock**
  - self-referential undecidability interlock
  - `godel_lock = true` forces HOLD / undecidable label

- **AC-T / Anomalous Contrast Tripwire**
  - local gradient detector
  - if `|AC| > threshold`, force extra grounding or ops pass

- **PSS / Paradox Scar Shadow**
  - overlap between historical paradox scars and current shadow mass
  - high values imply minefield conditions and possible `888_HOLD`

- **APE / Akal Present Energy**
  - available cognitive/compute free energy for the current step

- **echoDebt**
  - exploration vs exploitation debt

- **shadow**
  - residual unmodeled risk mass

- **kappa_r**
  - amanah / reversibility score

- **G2**
  - governed intelligence composite, intended as trust scalar

## Proposed Eval / Autoresearch Contract

The source proposes a Karpathy-style loop:

- one primary scalar per tool suite
- binary checks underneath
- only keep changes that improve the scalar

Candidate primary scalar:

- **Governed Utility Score** `U`

Candidate factors:

- physics quality
- governance compliance
- economics (`EMV`, `NPV`)
- linguistic contract adherence

## Proposed Binary Eval Questions

For each tool output:

1. physics band present and materially correct
2. math band present with explicit metrics
3. linguistic anchor present and appropriate
4. governance floors respected, including Gödel lock when needed
5. output matches structured contract
6. economic reasoning present when the tool proposes plans/actions
