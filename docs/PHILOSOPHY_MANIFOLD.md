## Why Words Are Powerful: The Neuroscience Model

The word-power theory is grounded in four empirically validated mechanisms:

### 1. Neural Simulation
Language activates the **same brain circuits as real experience** (amygdala,
motor cortex, visual cortex). The brain cannot fully distinguish language from
lived experience.

```
words → neural simulation → emotional reaction → action
```

*Quotes that invoke this: scar, triumph, love*

### 2. Symbolic Compression
One symbol compresses enormous lived experience. The word "war" encodes death,
trauma, politics, history, economics, human suffering — millions of moments in
one token.

```
symbol → compressed world model
```

*Quotes that invoke this: wisdom, paradox, seal*

### 3. Coordination
Shared narratives (money, nations, law, religion, science) coordinate millions
of humans without physical contact. Harari's cognitive revolution: shared stories
are the mechanism by which humans build civilization.

```
shared narrative → collective belief → coordinated action → civilization
```

*Quotes that invoke this: power, triumph, seal*

### 4. Attention Steering
Language directs limited cognitive attention bandwidth. "Look at the moon." →
consciousness steered immediately, involuntarily.

```
word → attention vector → perception redirected → cognition changed
```

*Quotes that invoke this: paradox, wisdom, power*

### Resonance Density ρ

Each quote has a **resonance density** ρ ∈ [0, 1] that captures how densely
compressed and neurologically potent it is. A high-ρ quote (e.g., ρ=0.95) packs
centuries of meaning into a single sentence. `seal` quotes have the highest
baseline ρ (0.95); `power` quotes the lowest (0.75).

## Selection Algorithm

### Step 1: Convert Session State to Query Vector

```python
from core.philosophy.coordinates import session_state_to_coordinate

coord = session_state_to_coordinate(
    stage="666_HEART",
    delta_s=-0.3,
    p2=1.1,
    g_score=0.78,
    psi=0.85,
    kappa_r=0.72,
    active_floors=["F6", "F7"],
)
query_vector = coord.as_vector()
# → [0.65, -0.3, 1.1, 0.78, 0.85, 0.72]
```

### Step 2: k-d Tree Query

Query the k-d tree for the k=10 nearest neighbours:

```python
distances, indices = kd_tree.query(query_vector, k=10)
```

### Step 3: Resonance Scoring

For each candidate:

```
score = 1/(1 + d) × (1 + floor_affinity_bonus) × (1 + ρ × resonance_weight)
```

Where:
- `d` = Euclidean distance in 6D space
- `floor_affinity_bonus` = `0.20 × n_matched_floors`
- `ρ` = quote's resonance density
- `resonance_weight` = `0.15`

### Step 4: Return Best Quote

The highest-scored candidate is returned with full provenance.
