- Read documents
- Use tools

- Fetch context

- Inspect environments

### What MCP Does NOT Do
MCP does NOT answer:
- Should this action be allowed?
- Does it violate human authority?
- Is confidence too high for weak evidence?

- Can we prove the action respected constraints?
- Should this be HOLD / VOID / SABAR / SEAL?

