# Noun Map

Canonical vocabulary for arifOS active surfaces.

## Charter
Authored declarative truth.
Examples: federation, deployment, sovereignty, kernel capability, tool charter.

## Registry
Enumerated machine-facing listing.
Examples: `tool_registry.json`, `registry.json`.

## Card
Compact discovery/profile document for clients or agents.
Examples: `server-card.json`, `agent.json`.

## Contract
Executable or normative agreement on behavior, fields, thresholds, or authority.
Examples: continuity contract, vault seal contract, risk contract.

## Manifest
Use only when the external protocol or the execution domain genuinely requires that term.
Examples: forge execution manifests, `agent-manifest/v1`, WebMCP/Apps protocol-specific language.

## Manifesto
Human philosophical or cultural declaration.
Examples: `000_MANIFESTO.md`, GEOX manifesto, human manifesto seal.

## Rule
If the file is authored system truth under arifOS control, prefer `charter`.
If the file exists for protocol compatibility, keep the protocol noun.
